
# Vaccine COVID-19

Landing page desarrollada como práctica de HTML y CSS. El objetivo fue replicar un diseño enfocado en promover la vacunación contra COVID-19 utilizando únicamente tecnologías web básicas, poniendo en práctica conceptos de maquetación, Flexbox and CSS Grid.


![App Screenshot](https://github.com/dsan-io/vaccine_kataCSS/blob/main/assets/vaccineOrginalKata.webp)


## Features

- Responsive Design
- Layout con Flexbox y Grid
- Diseño moderno
- Navegación sencilla
- Organización modular del CSS

## Tecnologías
- HTML5
- CSS3
- Flexbox
- CSS Grid

## Run Locally

Clone the project

```bash
    git clone https://github.com/dsan-io/vaccine_kataCSS.git
```

Go to the project directory

```bash
  cd vaccine_kataCSS
```

Después abre index.html en tu navegador.
